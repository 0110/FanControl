# Hardware
* ESP8266 in variant ESP-12 or ESP-07
* FAN of a computer with four PINs:
** 12V (red)
** GND (black)
** Tacho (yellow cable)
** PWM (blue cable)

# Software
* Arduino 1.6.12
** [EPS8266 Plugin 2.3.0](https://arduino-hannover.de/2015/04/08/arduino-ide-mit-dem-esp8266/)
